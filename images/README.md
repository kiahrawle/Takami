# Images — TAKAMI Guidebook

Save your images into this folder with exactly these filenames:

| Filename | Which image from the conversation |
|---|---|
| `logo.png` | TAKAMI logo (white background, navy face silhouette) |
| `hero-bottle.jpg` | Wide cinematic product shot — blue bottle with water bubbles, blue background |
| `bottle-simple.jpg` | Simple blue bottle on linen/white flat lay |
| `products-lineup.jpg` | 4 products side by side + 2 sheet masks |
| `products-shelf.jpg` | Products arranged on white shelf |
| `water-drop.jpg` | Blue water droplet ripple photo |
| `feather.jpg` | White feather on blue sparkly background |
| `pearl-woman.jpg` | Woman with pearl earrings, white fabric, minimal |

All images should be JPG except logo.png (PNG with transparency preferred).
Minimum recommended size: 1200px wide for photos, 300px for logo.
